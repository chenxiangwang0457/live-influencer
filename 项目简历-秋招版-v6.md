# 项目经历

**DeerFlow · AI 超级智能体系统** | 核心后端开发 | Python 3.12 / FastAPI / LangGraph / LangChain / SQLAlchemy

企业级 AI Agent 框架，29 层中间件链、20+ 公共 Skill。落地场景：**品牌方输入选人需求，系统自动采集多平台数据，经多维评分与多Agent并行分析后生成对比推荐报告，合作反馈驱动模型自优化，形成"搜索→分析→推荐→反馈进化"闭环。**

---

**1. 多智能体协作与反思。** 报告生成需覆盖粉丝画像、内容风格、商业表现、风险审查四个维度，单一Agent串行耗时长。基于LangGraph构建 Lead Agent 调度 + 子Agent并行执行架构——**SubagentLimitMiddleware 控制并发上限**，三路并发分析、自动触发风险扫描，Lead Agent 交叉校验后合成推荐报告；子Agent预算耗尽时评估已有产出质量决定重用或重试；Goal Evaluator 判定任务进展，连续无进展自动终止。**将人工2-3天压缩至分钟级，单次对比6位达人并输出结构化报告，子Agent推理可追溯率100%。**

**2. 记忆与知识管理。** 品牌方偏好和历史合作评价跨会话丢失，生成报告前需重复沟通。构建三层结构化记忆——用户画像、历史背景、带置信度的分类事实库——经 **MemoryMiddleware 防抖队列异步提取并原子写入**，配合保鲜机制自动淘汰过时偏好；已加载的领域 Skill 知识持久化复用，**DynamicContextMiddleware 在每轮请求中将记忆注入为独立数据块**，新会话自动继承。**运营人员可直接在新会话中继续未完成的报告任务，无需重复交代背景和偏好。**

**3. 上下文管理。** 报告生成涉及多份数据、子Agent中间结果和迭代修改，上下文快速膨胀。**SummarizationMiddleware 基于 token数/消息条数/模型容量三种OR触发渐进式摘要**，旧消息压缩为结构化摘要并保留最近对话窗口，AI/Tool消息对维持完整不拆分；**DurableContextMiddleware 在摘要前将委派记录和Skill引用捕获为持久上下文**，**SystemMessageCoalescingMiddleware 合并冗余 SystemMessage** 减少前缀缓存失效。**支持单会话50+轮深度分析不触发上下文溢出，多轮修改后关键结论不丢失。**

**4. 工具与Skill生态。** 不同品牌方需调用不同数据平台API和定制分析逻辑，修改框架代码不可维护。设计声明式SKILL.md扩展方案，支持即时激活和延迟发现双模式，7大类安全规则安装前阻断危险Skill；API密钥请求级带外注入、全链路6面封堵；MCP协议统一接入第三方数据源，**ToolErrorHandlingMiddleware 将工具异常归一化为可恢复消息**。**品牌方可自助配置专属分析Skill和报告模板，异常不中断报告生成。**

**5. 容错与稳定性保障。** 多平台API超时、子Agent崩溃、LLM幻觉覆盖报告内容等因素可能造成整次任务白费。构建五层纵深防线——**ToolErrorHandlingMiddleware 异常归一化 → ToolProgressMiddleware 停滞守卫逐级阻断 → ReadBeforeWriteMiddleware 文件写入门控防幻觉覆盖 → LoopDetectionMiddleware 重复调用硬停止** → 业务分级降级（部分数据源不可用输出中性分，全失败降级为纯数据对比），辅以**SandboxAuditMiddleware Shell命令分级审计**。**实现"宁给部分报告，不给空白页"，单点故障不级联。**

## 涉及的中间件（按出现顺序）

| 中间件 | 作用 | 切入点 |
|--------|------|--------|
| **SubagentLimitMiddleware** | 截断超出并发上限的 task 调用 | 多Agent协作 |
| **MemoryMiddleware** | 对话→记忆提取→防抖队列→原子写入 | 记忆管理 |
| **DynamicContextMiddleware** | 每轮注入当前日期+记忆到请求 | 记忆管理 |
| **SummarizationMiddleware** | OR触发式上下文压缩 | 上下文管理 |
| **DurableContextMiddleware** | 委派记录+Skill引用在摘要前持久化 | 上下文管理 |
| **SystemMessageCoalescingMiddleware** | 合并 SystemMessage 减少缓存失效 | 上下文管理 |
| **ToolErrorHandlingMiddleware** | 工具异常归一化为可恢复 ToolMessage | 容错 |
| **ToolProgressMiddleware** | 连续无新信息调用逐级阻断 | 容错 |
| **ReadBeforeWriteMiddleware** | 文件写入前校验读取标记防幻觉覆盖 | 容错 |
| **LoopDetectionMiddleware** | 检测重复工具调用模式硬停止 | 容错 |
| **SandboxAuditMiddleware** | Shell命令执行前审计 | 容错 |
